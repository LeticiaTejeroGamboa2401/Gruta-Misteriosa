# 🎮 La Gruta Misteriosa

Videojuego desarrollado con **Godot Engine** como parte del **Servicio Social**, con el objetivo de aplicar conocimientos de programación, diseño de videojuegos y desarrollo de software, así como aportar una solución lúdica con valor educativo y/o social.

---

## 🧩 Descripción del proyecto

Este proyecto consiste en un videojuego creado en Godot que integra mecánicas de juego interactivas, una narrativa básica y sistemas funcionales (movimiento, colisiones, puntuación, niveles, etc.). El desarrollo se realizó siguiendo buenas prácticas de programación y organización de proyectos.

El videojuego fue desarrollado durante el periodo de **Servicio Social**, cumpliendo con los objetivos académicos y formativos establecidos por la institución.

---

## 🎯 Objetivos

### Objetivo general

Desarrollar un videojuego funcional utilizando Godot Engine que demuestre habilidades técnicas, creativas y de gestión adquiridas durante la carrera universitaria.

### Objetivos específicos

* Aplicar programación en **GDScript**.
* Implementar mecánicas básicas de un videojuego.
* Diseñar niveles y escenas interactivas.
* Gestionar un proyecto de software utilizando control de versiones (Git).
* Documentar adecuadamente el desarrollo del sistema.

## ⚙️ Instalación y ejecución

1. Clona este repositorio:

   ```bash
   git clone https://github.com/LeticiaTejeroGamboa2401/Gruta-Misteriosa.git
   ```

2. Abre **Godot Engine**.

3. Selecciona **Importar proyecto**.

4. Elige la carpeta del proyecto.

5. Ejecuta el juego desde el editor (▶️).
